## 歡迎! 👋

這是使用 **Hugo** 建立的靜態部落格,具有以下功能:

### 主要功能

1. **全文搜尋** 🔍
   - 使用 Fuse.js 實現快速的客戶端搜尋
   - 支援標題、內容、摘要的搜尋
   - 中文搜尋完全支援

2. **標籤系統** 🏷️
   - 每篇文章可以添加多個標籤
   - 點擊標籤可以查看相關文章
   - 標籤雲展示

3. **評論功能** 💬
   - 整合 Giscus (基於 GitHub Discussions)
   - 或使用 Utterances (基於 GitHub Issues)
   - 完全免費且易於管理

4. **Markdown 支援** 📝
   - 完整的 Markdown 語法支援
   - 程式碼高亮
   - 數學公式支援 (可選)

### 程式碼範例

```python
def hello_world():
    print("Hello, Hugo Blog!")
    return "歡迎使用 Markdown"

hello_world()
```

```javascript
const greet = (name) => {
  console.log(`你好, ${name}!`);
};

greet("世界");
```

### 其他特色

- ✅ 響應式設計,支援手機、平板、桌面
- ✅ 深色模式支援
- ✅ SEO 優化
- ✅ RSS 訂閱
- ✅ 快速載入速度
- ✅ GitHub Pages 部署

### 如何使用

1. 在 `content/posts/` 目錄下建立新的 `.md` 文件
2. 添加 Front Matter (文章元數據)
3. 使用 Markdown 撰寫內容
4. 推送到 GitHub,自動部署!
